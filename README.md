# dorar-tafseer-by-surah

سكربت Python لاستخراج **موسوعة التفسير** من [dorar.net/tafseer](https://dorar.net/tafseer) وحفظها بترتيب السور — سورة واحدة لكل ملف.

## الهيكل

```
dorar_tafseer/
  001_سُورَةُ_الفَاتِحَةِ.md
  002_سُورَةُ_البَقَرَةِ.md
  003_سُورَةُ_آل_عِمْرانَ.md
  ...
  114_سُورَةُ_النَّاسِ.md
```

## بنية كل ملف

```
# سُورَةُ البَقَرَةِ

## تعريف السورة
أسماء السورة، فضائلها، المكي والمدني، مقاصدها، موضوعاتها...

---

## سُورَةُ البَقَرَةِ الآيات (1-5)
> https://dorar.net/tafseer/2/1

المعنى الإجمالي، غريب الكلمات، الفوائد البلاغية... [^1]

[^1]: يُنظر: ((تفسير ابن كثير)) (1/15).

---
```

## المميزات

- تعريف كامل لكل سورة (الأسماء، الفضائل، المقاصد، الموضوعات)
- الآيات القرآنية بين ﴿ ﴾
- الأحاديث النبوية بين « »
- حواشي مرقّمة مجمّعة أسفل كل ملف
- ترقيم تسلسلي فريد للحواشي في كل ملف (لا تداخل عند فتحه في Word)
- تخطي السور المحفوظة مسبقاً عند إعادة التشغيل

## المتطلبات

```bash
pip install requests beautifulsoup4
```

## الاستخدام

### عبر GitHub Actions (موصى به)
1. افتح تبويب **Actions**
2. اختر **Scrape Tafseer**
3. اضغط **Run workflow**
4. اختياري: أدخل عدد السور للاختبار (مثلاً `3`)

### محلياً
```bash
python scraper.py
```

لاختبار أول 3 سور فقط:
```bash
TEST_SURAHS=3 python scraper.py
```

## التحويل إلى Word

باستخدام [Pandoc](https://pandoc.org) مع قالب عربي:

```bash
pandoc "001_سُورَةُ_الفَاتِحَةِ.md" -o "الفاتحة.docx" --reference-doc=template.docx
```

أو لتحويل كل الملفات دفعة واحدة:

```bat
for %%f in (*.md) do pandoc "%%f" -o "%%~nf.docx" --reference-doc=template.docx
```
